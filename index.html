Code Day LA





A big thank you to everyone who attended Code Day LA!
Code Day LA is a 24-hour event anyone can attend to improve their coding skills and have fun.
Held at Toolbox, the event featured a basic coding lesson for beginners, a League of Legends tournament, delicious food, helpful volunteers and lots of awesome participants. In addition, there was a nationwide Code Cup competition which Los Angeles placed fourth in.
The event organizers also hosted a general coding competition with prizes including MVP, Best Presentation, Best Game, Best Application and Best Overall.



Congratulations to Coding Club member Aditya Bahawal for winning Best Presentation with his fun Unity game, Disco Snake.



Also, the game Hard Wrap, created by officers Sandra Tang and Stella Saw as well as Arcadia High alumni David Tang, won Best Game. Play the game here.
